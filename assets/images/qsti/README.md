# QSTI images

## Overview

Different images are used for the QSTI variant than for the base CTI image.
The convention is that images meant for the QSTI only are placed into this
`qsti` directory.

## Images

The existing images are their use are detailed here.

### background_1080p.png
In QSTI builds, this image is used as the background for demolauncher when the
display is configured to a resolution of 1080p (1920x1080).

### background_720p.png
In QSTI builds, this image is used as the background for demolauncher when the
display is configured to a resolution of 720 (1280x720).

### qnx_logo.png
A scaled QNX logo used by screewm in QSTI builds to brand its taskbar.
